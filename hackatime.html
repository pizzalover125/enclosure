Onshape Hackatime Guide

  • 1. Install the Onshape Wakatime browser extension
  • 2. Go to https://hackatime.hackclub.com/ and create an account.
  • 3. Follow the instructions on the Hackatime website to link your Onshape account.
  • 4. Click on settings.
  • 5. Copy API Key, as highlighted.
  • 6. Paste the API Key into the Wakatime settings. Also check the "Enable Tracking" checkbox!
