Requirements

  • Submission is a case of a phone or a calculator!
  • Any CAD software is allowed that has a viewable link with a history. This is to make sure you don't just import a case from Thingiverse. We highly, HIGHLY reccomend Onshape.
  • Submission is unique, personal, and fun!
  • Must have project files in a GitHub repository.
